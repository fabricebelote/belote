Importer / Exporter le Tournoi

Vous pouvez exporter les données du tournoi et créer un fichier XML. Pourquoi faire ça ? Soit vous souhaitez garder une copie du tournoi pour vos archives, soit vous avez un souci avec l’ordinateur et devez en changer au cours du tournoi.

Pour exporter le tournoi :

  1. Copiez les données dans le Presse Papier
  2. Sortez du mode plein écran - Touche F11
  3. Ouvrez un Éditeur de Texte
  4. Collez le Presse Papier
  5. Enregistrez le fichier avec suffixe ".xml"
  6. Revenez au mode plein écran - Touche F11

Pour importer le tournoi :

  1. Avant d'ouvrir cet outil...
  2. Ouvrez le fichier xml et copiez les données dans le Presse Papier
  3. Revenez ici...
  4. Collez les données dans la case rouge
  5. Cliquez sur le bouton « Importez »
